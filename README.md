## 📌 Project Overview
A full-stack CRUD application built using **Django REST Framework** and **React (Vite)** with **Docker containerization** and **CI/CD pipeline using GitHub Actions**.

This project demonstrates modern **DevOps practices** including containerization, automated builds, and continuous integration.

---

## 🧠 Tech Stack

| Layer | Technology |
|------|-------------|
| Frontend | React + Vite |
| Backend | Django + Django REST Framework |
| Database | SQLite |
| Containerization | Docker |
| CI/CD | GitHub Actions |

---

## 🏗 Project Architecture

React UI → REST API → Django Backend → Database


docker compose up --build


Access:


Frontend → http://localhost:3000

Backend → http://localhost:8000

Admin → http://localhost:8000/admin


---

## 🔄 CI/CD Pipeline

Pipeline automatically runs when code is pushed.

Steps:

1. Install dependencies
2. Run backend tests
3. Install frontend packages
4. Build Docker containers
